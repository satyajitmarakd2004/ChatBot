# 🤖 Java ChatBot

A simple Java-based chatbot that interacts with users using predefined responses. This project demonstrates Java fundamentals, OOP concepts, user input handling, and conditional logic.

## ✨ Features
- Interactive chat
- Predefined responses
- Continuous conversation
- Simple and clean code

## 🛠️ Technologies
- Java
- OOP
- Collections
- Exception Handling

## ▶️ Run
1. Clone the repository.
2. Open in Eclipse or IntelliJ IDEA.
3. Run `Main.java`.
